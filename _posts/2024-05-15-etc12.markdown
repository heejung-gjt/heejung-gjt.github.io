### eval()
eval 함수는 매개변수로 받은 식을 문자열로 받아서 실행하는 함수이다.    
즉, 매개변수로 받은 식은 실행 가능한 문자열로 들어와야 한다   

```python
a = '1 + 3 + 4 + 5'
eval(a)  # 13

a = '1 + 3 + '
eval(a)  # SyntaxError: invalid syntax
```
