### dict.fromkeys(seq, value)
딕셔너리 생성할 때 사용하는 메소드이다
- seq: 생성하려는 딕셔너리 키의 목록
- value: 생성하려는 딕셔너리 값

```python
seq = 'test'
dic1 = dict.fromkeys(seq)
print(dic1)  # {'t':None, 'e':None, 's':None, 't':None}

dic2 = dict.fromkeys(seq, 0)
print(dic1)  # {'t':0, 'e':0, 's':0, 't':0}
```
