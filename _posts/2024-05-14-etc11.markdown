### oct()
- 10진수 -> 8진수 문자열
```python
a = '10'
oct(int(a))  # '0o12'
```

- 8진수 -> 10진수
```python
a = '10'
int(a, 8)
```

<br>


### hex()
- 10진수 -> 16진수 문자열
```python
a = '10'
hex(int(a))  # '0xa'
```

- 16진수 -> 10진수
```python
a = '10'
int(a, 16)
```

<br>


### bin()
- 10진수 -> 2진수 문자열
```python
a = '10'
bin(int(a))  # '0b1010'
```

- 2진수 -> 10진수
```python
a = '10'
int(a, 2)
```
